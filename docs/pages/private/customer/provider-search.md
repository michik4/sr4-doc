# **Поиск Поставщиков**
## **Резюме**
--- 
Поиск поставщиков - это полноценный поисковой модуль, находящий поставщиков (компаний) по множеству фильтров и сортировок. 
### **Поисковой запрос**
Поисковой запрос строится из множества фильтров, сортировок и полей. 
#### Фильтры
В фильтры входят:

- **По выручке** (от `x` до `y`)
- **По категории работ** (множественный выбор из древовидного списка)
- **По рейтингу** (не ниже н-ного кол-ва)
- **По регионам** (множественный выбор из плоского списка)
- **Наличие СРО допусков** (не ниже н-ного уровня из 5)
	- ПИР
	- СМР
- **Наличие лицензий**
	- МЧС
	- ФСБ
	- МИНКУЛЬТ
- **Наличие подписки** (да/нет)
#### Сортировки
В сортировки входят:

- По рейтингу 
- По выручке
- По дате обновления на платформе
- По дате регистрации на платформе
- По кол-ву отзывов
