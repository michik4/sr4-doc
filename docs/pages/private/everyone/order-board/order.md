# **Страница заказа**
##  **Резюме**
---
Страница на которой отображается полная информация по заказу, вместе с кнопками взаимодействия

## **Блоки**
---
### Информация о заказе
Выводит всю информацию о [заказе](../../../../entities/order-board/order.md#full-data)

### Кнопки взаимодействия
В них входят быстрые действия:

- **У Заказчика** (автора заказа)
    - Новый
        - Открыть отклики
        - Завершить сбор откликов
        - Отменить заказ
    - В работе
        - Открыть профиль исполнителя
        - Чат 
        - Отменить заказ
    - Выполнен
        - Оставить отзыв о Поставщике
    - Отменен/просрочен
        - *кнопки отсутствуют*
- **У Поставщика**
    - Новый 
        - Откликнуться
        - Пожаловаться
    - В работе
        - Чат
        - Открыть профиль Заказчика
    - Выполнен/отменен
        - Запросить отзыв
