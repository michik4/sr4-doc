# **Биржа заказов**
## **Резюме**
---

Служит для размещения заказов (запросов) Заказчиками и откликов на них, со стороны Поставщиков.

## **Блоки**
---

### **Блок-заголовок**
Заголовок с текстом "Биржа заказов". 
> Можно разместить несколько кнопок быстрых действий, по типу "создать заказ", или "Посмотреть отклики"

### **Строка с вкладками**
Несколько вкладок для удобного разделения и поиска функций:

- **Для Заказчика**
    - Все заказы
    - Мои заказы
    - Новые отклики
- **Для Поставщика**
    - Все заказы
    - Мои отклики

### **Строка поиска** (все заказы) / **фильтр статусов** (мои заказы, мои отклики)
Строка для ввода поисковых запросов для фильтрации заказов / набор фильтров для выдачи (по статусам).
> Возможно, для вкладки "все заказы" лучше будет здесь же вывести список фильтров (горизонтально), нежели выводить их в боковое меню. Стоит поэкспереметировать. 

### **Выдача** (в зависимости от вкладки)
Список результатов поиска (карточки, строки)

- [Заказы](../../../entities/order-board/order.md)
- [Отклики](../../../entities/order-board/response.md)

### **Боковое меню**

#### Вкладка "все заказы"
В нем находятся список фильтров для поискового запроса:

- По дате создания
- Стоимости (от/до)
- По категориям
- По регионам
