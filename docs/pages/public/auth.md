# **Аутентификация**
## **Резюме**
---
Страница для регистрации, или аутентификации пользователей. Для разных ролей разная система регистрации, но система аутентификации общая.
### Виды регистрации
#### Для поставщиков
Для поставщиков доступна форма регистрации. Верификация, кроме почты, не требуется.
#### Для заказчиков
Регистрация заказчика происходит через выдачу админ-ом пригласительной ссылки. По переходу по ссылке, заказчику показывается такая же, как и у Поставщика, форма регистрации.
#### Для администраторов/модераторов
Регистрация админов происходит через создания учетных записей главным админ-ом. Формы регистрации как таковой нет. 
## **Блоки**
--- 
### **Форма регистрации (для Поставщиков)**
#### Ввод данных
Данные для регистрации:

- Почта
- Пароль
- ИНН
#### Подтверждение регистрации (через почту)
На почту высылается письмо с ссылкой подтверждения, нажатие на которую переводит на страницу подтверждения, после которой можно будет пользоваться учетной записью. 
### **Форма регистрации (для Заказчиков, скрытая)**
#### Ввод данных
Данные для регистрации: 

- Почта
- Пароль
#### Подтверждение регистрации (через почту)
На почту высылается письмо с ссылкой подтверждения, нажатие на которую переводит на страницу подтверждения, после которой можно будет пользоваться учетной записью. 
### **Форма аутентификации**
Общая для всех пользователей. Для входа потребуется:
- Почта
- Пароль
