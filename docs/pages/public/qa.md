# <a id="qa-h1" style="text-decoration: none; color: black;"/>**Q/A (Вопросы и ответы)** 
## **Резюме**
--- 
Служит руководством по использованию платформы, отвечая на часто задаваемые вопросы. Дизайн должен быть схожим с [Правовой информацией](../legal-info#legal-info-h1).
## **Блоки**
---
### Боковое меню
В нем находятся список всех функций, которые могут вызвать вопросы. Также должны быть кнопки/ссылки:
- Задать вопрос (тех. поддержка)
- Поле с контактами для связи
### Текущий вопрос
В этом блоке находится описание текущего вопроса, или блока вопросов, объединенных по теме.
