# **Информация для Заказчиков (страница)**
## **Резюме**
--- 
Презентационная страница для Заказчиков. Ее главная задача - заинтересовать и направить пользователя на регистрацию (путем ссылки или формы).
## **Блоки**
---
### Герой-блок
Промо-блок, в котором через лозунги рассказываются преимущества платформы. Можно использовать статистику или другие цифры. 
### Список функционала
Перечень функционала для Заказчика на платформе.
### Балловая система
Описание балловой системы поощрений
### CTA-блок
Блок подводящий к регистрации. Должен привлечь пользователя на страницу аутентификации, после всего контента выше. Может быть в виде формы/ссылки.
