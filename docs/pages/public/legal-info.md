# <a id="legal-info-h1" style="text-decoration: none; color: black;"/>**Правовая информация**
## **Резюме**
---
Общая страница предоставляющая все нужные документы для пользования платформой. Стиль должен быть единым с [Q/A](../qa#qa-h1).
## **Блоки**
--- 
### Боковое меню
В нем должен находится список всех документов и кнопки для доп. действий:

- Скачать текущий документ
- Скачать все документы
- Обратиться в поддержку
- Поле с контактами
### Блок с текущим документом
В нем должны быть:

- Просмотр текущего, выбранного в боковом меню, документа
- Шапка с панелью инструментов 
	- название текущего документа (с ограничением по символам)
	- скачать текущий документ 
	- кнопки навигации 
		- следующий документ
		- прошлый документ
