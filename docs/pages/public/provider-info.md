# **Информация для Поставщиков (страница)**
## **Резюме**
--- 
Презентационная страница для Поставщиков. Ее главная задача - заинтересовать и направить пользователя на регистрацию (путем ссылки или формы).
## **Блоки**
---
### Герой-блок
Промо-блок, в котором через лозунги рассказываются преимущества платформы. Можно использовать статистику или другие цифры. 
### Список функционала
Перечень функционала для Поставщика на платформе.
### Тарифы
Описание тарифов и их преимуществ.
### CTA-блок
Блок подводящий к регистрации. Должен привлечь пользователя на страницу аутентификации, после всего контента выше. Может быть в виде формы/ссылки.
