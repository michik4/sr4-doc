# Памятка: Сведения о компании (Карточка предприятия)

1. Основная информация об организации
	1. Полное юридическое наименование + ИНН
	2. Организационно-правовая форма (ООО, АО, ГУП и т.д.)
	3. Сокращенное юридическое наименование
	4. Регистрационные данные
		- Дата регистрации
		- Место регистрации
		- Орган регистрации
	5. История наименования (если менялось)
		- Прежнее наименование
		- Дата изменения
	6. Руководство (Исполнительный орган)
		- Должность (Ген. директор, Директор и др.)
		- ФИО (полностью)
		- Телефон
		- E-mail
	7. Главный бухгалтер
		- ФИО (полностью)
	8. Структура владения
		- Учредители с долей >10% (ФИО)
	9. Финансовые параметры
		- Размер уставного капитала
		- Профилирующее направление деятельности
		- Режим налогообложения (ОСНО с НДС / УСН без НДС)
	10. Идентификаторы и коды
		- ОГРН
	11. Банковские реквизиты
		- Расчетный счет (основной)
		- Наименование банка, БИК, к/с

2. Контактная информация
	1. Адреса
		- Юридический (с индексом)
		- Фактический (с индексом)
	2. Связь
		- Телефоны (офис, секретариат)
		- Факс
		- E-mail (общий/тендерный)
		- Адрес веб-сайта

3. Квалификация и допуски
	1. Членство в СРО
		- Название СРО
		- Реквизиты решения о приеме (дата, номер протокола)
		- Описание разрешенных работ (уровни ответственности)
	2. Сертификация
		- ISO (наличие, номер сертификата, описание)
	3. Структура компании
		- Филиалы и обособленные подразделения
		- Дочерние предприятия
	4. Материально-техническая база (МТБ)
		- Собственные мощности (площадь, адрес)
		- Арендованные мощности (площадь, адрес)
		- Спецтехника (собственная/арендованная)

4. Финансовые показатели (Обороты)
	1. Выручка за текущий год (с нарастающим итогом)
	2. Выручка за прошлый год (год N-1)
	3. Выручка за позапрошлый год (год N-2)
	4. Выручка за год N-3

5. Кадровый состав
	1. Динамика численности (среднесписочная)
		- Текущий год
		- Предыдущие 3 года
	2. Структура персонала (кол-во человек)
		- Руководящий состав
		- ИТР (Инженерно-технические работники)
		- Рабочие
		- Квалификация (стаж работы в фирме >5 лет)

6. Опыт и компетенции (Основные направления)
	1. Проектирование
		- Жилые, медицинские, образовательные, спортивные объекты, офисы.
		- Инженерные сети (вода, тепло, свет, газ, слаботочка).
	2. Строительно-монтажные работы (СМР) - Генподряд
		- Соцкультбыт, жилье, коммерческая недвижимость.
	3. СМР - Инженерные сети и сооружения
		- Котельные, очистные, наружные сети.
	4. Специализированные работы
		- Дороги, благоустройство.
		- Отделка, фасады, кровля.
		- Свайные, бетонные, земляные работы.
		- Монтаж металлоконструкций, лифтов.
		- Пусконаладка (ПНР).
	5. Поставки оборудования
		- Комплексное оснащение (школы, больницы).
	6. Услуги
		- Технический заказчик.
		- Строительный контроль.
		- Экспертиза, изыскания.

7. Портфолио (Реестр проектов)
	1. Данные по ключевым объектам
		- Вид работ
		- Заказчик
		- Роль (Генподрядчик/Субподрядчик)
		- Сроки исполнения
		- Стоимость контракта
		- Объем работ (физический)

8. Партнерская сеть
	1. Основные субподрядчики
		- Наименование, ИНН
		- Виды передаваемых работ

9. Пакет уставных документов (Чек-лист наличия сканов)
	1. Свидетельства (ОГРН/Лист записи, ИНН)
	2. Устав (актуальная редакция)
	3. Документы на Ген. директора (Решение/Протокол)
	4. Приказ на Главбуха
	5. Бухгалтерская отчетность (Ф1, Ф2 с отметкой ФНС)
	6. Справки об отсутствии задолженности (КНД 1120101)
	7. Выписка из СРО (актуальная)
	8. Документы на офис (Аренда/Собственность)

# Список данных, которые МОЖНО получить из API (Базовая лицензия)

### 1. Основная информация об организации
- **Полное и сокращенное юридическое наименование** (поля `company_full_name`, `company_short_name`).
- **ИНН, ОГРН, ОКПО, КПП** (включая дополнительные КПП филиалов).
- **Организационно-правовая форма** (текстовое описание, например "Публичные акционерные общества").
- **Регистрационные данные**: Дата регистрации, Регион, Краткое наименование регистрирующего органа.
- **Руководство (Ген. директор)**: ФИО, должность, ИНН директора.
- **История руководителей**: Список прежних директоров с датами.
- **Структура владения**: Список учредителей (ФИО/Название), их ИНН, размер доли (в % и рублях), даты входа.
- **Уставной капитал**: Размер в рублях.
- **Основной вид деятельности**: Код и расшифровка основного ОКВЭД + список дополнительных кодов.
- **Банковские реквизиты (частично)**: Наименование банка, БИК, Корр. счет (расчетный счет часто скрыт).

### 2. Контактная информация
- **Адреса**: Юридический, Фактический (разбитые на составляющие: индекс, улица, дом).
- **Связь**: Общий список телефонных номеров и Email-адресов (без разделения на отделы).
- **Веб-сайты**: Список доменов компании.

### 3. Квалификация и допуски
- **Лицензии (Ограничено)**: Только 3 последних актуальных документа (номер, дата, кем выдан, описание деятельности).
- **Связанные лица**: Список дочерних компаний (где организация является учредителем) и филиалы (через мульти-КПП).

### 4. Финансовые показатели (Бухгалтерская отчетность)
- **Выручка (код 2110)**: Годовые значения за текущий (если сдан отчет) и прошлые годы.
- **Стоимость основных средств (МТБ)**: Общая балансовая стоимость (код 1150).
- **Чистая прибыль/убыток**: Годовые значения.
- **Баланс**: Сумма активов и пассивов.

# Список данных, которые НЕЛЬЗЯ получить из API (Базовая лицензия)

### 1. Ограничения Базовой лицензии (данные есть в базе, но скрыты тарифом)
- **Система налогообложения**: Нельзя узнать, работает ли компания на ОСНО (с НДС) или УСН.
- **Кадровый состав**: Количество сотрудников, средняя зарплата, динамика численности.
- **Полный список лицензий**: Недоступна история всех лицензий (более 3-х штук).
- **Анализ конкурентов**: Сравнение выручки с конкурентами.
- **Финансовый анализ**: Текстовые описания рентабельности и эффективности (расширенная аналитика).

### 2. Отсутствуют в API (нет методов или полей)
- **История наименований**: Прежние названия компании.
- **Главный бухгалтер**: ФИО и контакты.
- **Детальные банковские счета**: Номер основного расчетного счета (часто приходит `null`).
- **Членство в СРО**: Номера протоколов, уровни ответственности, взносы в комп. фонды.
- **Сертификаты ISO**: Отдельного поля нет (только если попадет в общие лицензии).
- **Детализация МТБ**: Площадь офисов/складов, список конкретной спецтехники, наличие аренды.
- **Структура персонала**: Разделение на ИТР и рабочих, стаж сотрудников.
- **Портфолио и Проекты**: Списки выполненных объектов, генподряд/субподряд, заказчики по конкретным стройкам.
- **Партнерская сеть**: Списки субподрядчиков и поставщиков.
- **Сканы документов**: API отдает только данные (JSON), получить PDF/JPG сканы Устава, Решений, Свидетельств или балансов с печатями нельзя.
