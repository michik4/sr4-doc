# **Заказ**

## **Поля**

### <a id="short-data" style="text-decoration: none; color: black;"/> **Краткие данные**

- **Новый**
	- Название заказа
	- Прием откликов до (дата)
- **Отменен/Просрочен**
	- Название заказа
	- Дата отмены/просрочки
- **В работе**
	- Название заказа
	- Отсчет времени (в днях, на основе срока выполнения, из отклика)
- **Выполнен**
	- Названия заказа
	- Дата завершения
### <a id="full-data" style="text-decoration: none; color: black;"/> **Все данные**
Используются при создании заказа и на отдельной странице заказа.

* **Название заказа** (Текстовое поле, обязательно)
* **Описание** (Текстовая область, обязательно)
* **Категории (необязательно)** (Выбор из списка)
* **Начальная минимальная цена** (Текстовое/Числовое поле)
* **Прием Откликов до** (Дата)
* **Срок оказания услуги** (Дата)
* **Регионы выполнения работ** (Выбор из списка)
* **Контактный телефон** (Поле для ввода телефона)
* **Контактный email** (Поле для ввода email)
* **Организатор торгов** (Текстовое поле)
* **Условия оплаты** (Текстовая область)
* **Место поставки** (Текстовое поле)
* **ТЗ в PDF** (Загрузка файла .pdf)
* **Форма КП в Excel** (Загрузка файла .xlsx, .xls)
* **Ссылка на документацию** (Поле для ссылки)
