# **Отклик**

## **Поля**

### **Краткие данные**
Краткое описание используется в карточках откликов (наприм. в ЛК Поставщика "мои отклики")

> Нужно поиграться с наполнением карточки и входящими в нее данными.

- **На рассмотрении/отказ**
	- Заказ
	- Цена
- **Принят** 
	- Заказ
	- Цена
	- Отсчет времени (в днях, на основе срока выполнения)
- **Завершен**
	- Заказ
	- Цена
	- Дата завершения
### **Все данные**
Все данные используются на отдельной странице отклика, для отображения всей информации, или в форме создания заказа.

- **Заказ** (Заказ к которому относится отклик)
- **Описание предложения** (Текстовая область, обязательно)
- **цена (₽)** (Числовое поле)
- **Срок выполнения (дней)** (Числовое поле)
- **Портфолио и примеры работ** (Кнопка добавления работ)
- **Форма коммерческого предложения (Excel)** (Загрузка файла .xlsx, .xls, .xlsm)
- **Счет (PDF)** (Загрузка файла .pdf, необязательно)
- **Контактный телефон** (Поле для ввода телефона)
- **Email для связи** (Поле для ввода email)
